# Pull Request Template
PRテンプレートを管理するリポジトリです。

## 📁 ディレクトリ構成
```
.github/
  pull_request_template/
    feature.md
    bugfix.md
    refactor.md
```

### 📝 テンプレート内容
feature.md
```md
### 概要
このPRではどの機能を追加したかを記載してください。

## 変更内容
- 新規追加した処理や画面の概要を箇条書きで記載
- 影響範囲がある場合は併せて記載

## 動作確認
- 動作確認手順
- スクリーンショットなどあれば添付

## 関連Issue
Fixes #
```

bugfix.md
```md
## 概要
どのバグを修正したか、原因と対応内容を記載してください。

## 修正内容
- 修正箇所の概要
- なぜこの修正が必要だったか

## 再発防止策（任意）
- 気づきや改善点があれば記載

## 動作確認
- 再現手順と、修正後の確認結果

## 関連Issue
Fixes #
```

refactor.md
```md
## 概要
リファクタリングの目的と範囲を記載してください。

## 実施内容
- どの部分をどのように改善したか
- ロジック変更の有無

## 影響範囲
- 影響が考えられる箇所を記載

## 動作確認
- 動作に影響がないことをどう確認したか
```
